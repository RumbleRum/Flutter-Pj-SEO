[ 플러터 명령어 모음 ]
- 형식 : flutter 명령어

1. 플러터 프로젝트 팩키지 생성명령어
- flutter create 프로젝트명

2. flutter 모듈 실행 명령어
- flutter run (참고: 크롬 브라우저는 2번옵션)

3. 플러터 설치 진단명령어
- flutter doctor

4. 플러터 빌드 명령어 : 특정 플렛폼에 맞게 빌드
- flutter build   특정 플랫폼 명령어

5. 플러터 최신버전 업데이트
- flutter upgrade

6. 플러터 의존성 패키지 설치 명령어
- flutter pub get (전체설치)
- flutter pub add 팩키지명 (부분설치)
-> pubspec.yaml 파일에 의존성 설치 기록됨!
