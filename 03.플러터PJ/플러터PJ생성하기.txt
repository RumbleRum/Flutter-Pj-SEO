[ 플러터 PJ 생성하기 ]

-> 플러터 명령어를 통해 개발표준 패키지를 설치하여
개발을 더 쉽고 용이하게 할 수 있도록 구글이 지원한다.


-> 표준개발 프로젝트 패키지를 보일러플레이트 라고함!(boilerplate => 프로젝트 표준코드

[ 설치 순서 ]
1. 설치할 위치로 이동한다(cd명령어)
2. cmd 창에서 프로젝트 생성 플러터 명령어를 실행함
((프로젝트 명명법))
-> 숫자로 시작불가 , -  등 특수문자 사용불가
-> 일반적으로 언더바를 사용한 단어연결로 명명함!
예) app_01_store
->> flutter create app_01_store

3. 설치후 프로젝트 파일로 이동한다(cd명령어)

4. 프로젝트 폴더 위치에서 모듈을 실행시킨다
->> flutter run
-> 옵션중 2번 크롬브라우저 선택 - > 숫자키 2누르기
-> 모듈 싱행하여 일정시간 후 브라우저로 로딩됨!!!

5. cmd 창에서 ctrl + c  하면 닫겠냐고 물어봄 y하면 닫음
또는 그냥 쓰레기통 아이콘 클릭, 또는 브라우저 닫기
